Asta4D Sample Project - Index

Simple usages

  • Use Snippet for rendering dynamic page.
  • Inject values of the variable.(/app/{name}/{age})
    • name=foo, age=25
    • name=bar, age=20
  • Manipulate attribute values.
  • Extend other template.
    • append
    • insert
    • override
  • Embed other template.
  • Use Request Handler for side effects.

Complicated usages

  • form
  • Experience some type of Renderers.
  • Pass variables from outer Snippet to inner Snippet.
  • Use dynamic Snippet and dynamic embed.
  • Use ContextData annotation.
  • Localize Messages.
